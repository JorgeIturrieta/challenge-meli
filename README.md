### Para iniciar el proyecto seguir los siguentes pasos:

1- Iniciar backend .

`cd backend-challenge-meli`

`npm install`

`npm start`

2- Iniciar frontend.

`cd frontend-challenge-meli`

`npm install`

`npm start`
